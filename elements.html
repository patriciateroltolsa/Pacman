Milestone(Korean)

이 페이지는 한국어로 작성된 페이지입니다.
세종대학교 2017 오픈소스SW개론 4조의 진행 상황을 적고 있습니다.
기본적으로 회의는 매 강의 시간 전과 후에 짧게 이루어졌으며,
긴 회의가 필요한 경우 스터디룸을 예약하여 회의를 하였습니다.

Github 업데이트 사항은 팀원들에 의해 작성된 Issues와 최종적으로 master에 Merge된 Pull requests를 기반으로 작성하였습니다.
