    Topics / Purpose

  • - Topic

    We selected Pac-man as our topic, a classic video game. Probably most of people would have tried this game in childhood. We decided to remake the game into a modern style game. Pac-man was published in 1980 and the goal of the game is for a simple 2D character to eat some dots to get scores while avoiding being caught by spooky ghosts. Pac-man is one of the most famous games along with the Super Mario series of Nintendo. If you google Pac-man, you will see the picture above and you can play it on the web page by pressing the 'Game Start!' button. There is even a film like 'Pixels' which features classic video game characters including Pac-man. Recently Google also had a Pac-man game event on their Google Map for April Fool's Day. You can see that although the game is old and classic, it is still loved by people.


    대표적 고전게임이자 비디오 게임인 팩맨(Pac-Man)을 주제로 정하였다. 어릴 적 누구나 한 번쯤 해보았을 팩맨 게임을 직접 구현하여 현대적인 스타일로 바꾸어 본다. 팩맨은 1980년에 만들어졌으며 기존 게임의 경우 2D 형태로 팩맨 캐릭터가 도트를 먹으며 점수를 얻고 고스트를 피해 도망 다니는 규칙을 갖고있다. 팩맨은 닌텐도의 마리오 시리즈와 더불어 전 세계적으로 많이 알려진 게임이기도 하다. 또한 구글에서 팩맨을 검색하면 위의 사진과 같은 게임이 나타나게 되고 ‘게임 시작!’을 클릭하여 웹 페이지 상에서 팩맨을 즐길 수 있다. 이외에도 팩맨같은 고전 비디오게임을 주제로한 영화 ‘픽셀’과 최근 구글에서 만우절에 구글 맵에 팩맨을 적용한 이벤트를 했다고 하는데, 이처럼 고전게임 임에도 불구하고 아직까지도 많은 사랑을 받고있다.

    - Purpose

    We will fix the original project to remove several bugs and add proper functions of Pac-man. At the same time we will try to make the graphic seem bit more modern. We will change the 2D graphics into 3D, and display third and first person view in one window. Adding first person viewpoint will make the game more exciting than when it was only third person game. This will turn original Pac-man into a new game with fascinating graphics and stability. Besides that, we will create the environment for executing the project and document it. After revising the source code and adding new functions, we will suggest a pull request to the original maker of the project. If the maker accepts our request we will participate directly in the project. If he/she doesn't, we will fork the original project and make one of our own.

    기존 프로젝트에 팩맨의 기능을 버그 없이 추가하여 구현함과 동시에 팩맨의 그래픽을 현대적으로 구현한다. 그래픽 측면에서는 2D로 구현되어 있는 팩맨을 3D로 만들고, 3인칭과 1인칭 시점을 동시에 구현하여 기존의3인칭 시점에 없던 1인칭 시점의 생동감을 주는 등 기존의 팩맨보다 화려한 그래픽을 가진 오류 없는 프로젝트가 되도록 만든다. 이외에도 프로젝트 실행을 위한 환경을 구축하고 문서화 시킨다. 기존의 소스 코드를 다듬고 기존의 프로젝트에 위와 같은 기능을 추가하여 원작자에게 Pull Requests를 해서 받아들여지게 된다면 프로젝트에 직접 참여하고 거절되면 Fork하여 자체적 프로젝트를 만든다.
